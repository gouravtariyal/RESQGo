# assets

Static media for RESQGo. Keep binaries here; import via relative paths or a bundler alias (e.g. `@assets`).

| Folder | Purpose |
|--------|---------|
| `images/` | Raster / vector images (PNG, JPG, SVG) |
| `icons/` | App icons and UI glyph assets |
| `fonts/` | Custom typeface files |
| `animations/` | Lottie / animation JSON files |

Do not put screens or business logic in this folder.
